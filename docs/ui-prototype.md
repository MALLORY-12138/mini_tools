# Mini Tools Chrome 插件 UI 原型

## 1. 选中文本后的右键菜单

触发条件：用户在网页里选中任意文本并右键。

```text
┌──────────────────────────┐
│ 复制                     │
│ 搜索所选内容             │
├──────────────────────────┤
│ Mini Tools >             │
│   查地区                 │
│   查汇率                 │
└──────────────────────────┘
```

设计要点：
- 菜单只在 `selection` 场景出现，避免干扰普通右键菜单。
- 两个动作并列：`查地区` 用本地 CSV，`查汇率` 用实时 API。

## 2. 查地区结果面板

触发条件：选中 `international_code`、`country_or_region_zh`、`english_name` 或 `phone_code` 中任意一项。

```text
┌────────────────────────────────────┐
│ Mini Tools                    [×]  │
│ 查地区                             │
│ 正在查询：US                       │
├────────────────────────────────────┤
│ 1 条匹配结果                       │
│ ┌────────────────────────────────┐ │
│ │ 美国                           │ │
│ │ 国际代码        US   ← 高亮    │ │
│ │ 中文地区        美国           │ │
│ │ 英文名称        United States  │ │
│ │ 电话区号        +1             │ │
│ └────────────────────────────────┘ │
└────────────────────────────────────┘
```

设计要点：
- 面板固定在页面右上角，避免遮挡选中文本附近的原页面内容。
- 被选中的字段用浅绿色高亮，其他三项完整展示。
- 未匹配时显示空状态，提示可选中哪些字段。

## 3. 查汇率结果面板

触发条件：选中货币代码、货币名称、带金额的文本，或自然语言换算文本，例如 `usd`、`100 EUR`、`300法郎换算韩元`。

```text
┌────────────────────────────────────┐
│ Mini Tools                    [×]  │
│ 查汇率                             │
│ 正在查询：300法郎换算韩元          │
├────────────────────────────────────┤
│ ┌──────────────┐ ┌──────────────┐  │
│ │ 美元 -> 人民币│ │ 人民币 -> 美元│  │
│ │ 1 USD = ...  │ │ 1 CNY = ...  │  │
│ └──────────────┘ └──────────────┘  │
│                                    │
│ 换汇计算器                         │
│ 金额 [300________]                 │
│ 源货币 [CHF - Swiss Franc____] [⇄] │
│ 目标货币 [KRW - South Korean Won]  │
│ 300 CHF = ... KRW                  │
│                                    │
│ 300 CHF  Swiss Franc               │
│ 基准日期 2026-06-28               │
│ [筛选币种：eur____________]        │
│                                    │
│ EUR   Euro                 100     │
│ USD   US Dollar            ...     │
│ CNY   Chinese Yuan...      ...     │
└────────────────────────────────────┘
```

设计要点：
- 顶部优先展示 USD/CNY 双向汇率。
- 当前选中币种作为列表基准，支持金额换算。
- 计算器支持手动修改金额、源货币和目标货币，并可一键交换源/目标货币。
- 列表内支持搜索过滤货币代码或英文名称。
- API 请求失败时自动尝试 fallback URL，再尝试本地缓存。

## 4. 加载、错误和空状态

```text
加载中：   [spinner] 加载中...
空状态：   未找到匹配地区 / 可选中两位国际代码、中文地区名、英文名称或电话区号后重试。
错误态：   汇率查询失败 / 展示实际错误信息。
```

设计要点：
- 所有状态都在同一个右上角面板内完成，不打开新页面。
- 关闭按钮固定在面板头部，用户可随时收起。
